Deep Learning-Based System for Crop Pest and Weed Identification

Vision Transformer (ViT)-Based Deep Learning for Plant Health Classification

In precision agriculture, timely and accurate identification of pests and weeds is critical for maximizing crop productivity and minimizing economic losses. Traditional methods like manual inspection or rule-based image analysis are often labor-intensive, slow, and prone to human error.To address these challenges, I developed a deep learning solution leveraging a Vision Transformer (ViT) architecture for classifying plant images into categories such as pest-infected, weed-dominated, and healthy crops. Using a pretrained ViT-Base-16 model, I fine-tuned it on a labeled dataset with comprehensive data augmentation techniques—including color jitter, Gaussian blur, and random rotation—to enhance model robustness and generalization.The dataset was split 80/20 for training and testing, and the model was trained using the Hugging Face Trainer API with carefully optimized hyperparameters. The final model achieved a test accuracy of 91.77%, outperforming traditional and shallow-learning approaches.Unlike hardware-intensive alternatives, this ViT-based solution is cost-effective, scalable, and maintenance-free, enabling real-time plant health monitoring. This work demonstrates strong potential for integration into smart farming platforms, empowering farmers and agronomists with automated, data-driven decision-making tools—contributing to more sustainable and efficient agricultural practices.

Professional OWASP Top 10 Vulnerability Scanner

Python-Based Automated Web Application Security Assessment Tool

Modern web applications are continuously exposed to security threats, making regular vulnerability assessments essential for identifying and mitigating potential risks before they can be exploited. To address this challenge, I developed a professional Python-based vulnerability scanner that automates the detection of common web application security issues aligned with the OWASP Top 10 framework. The scanner performs comprehensive security testing by identifying vulnerabilities such as SQL Injection, Cross-Site Scripting (XSS), Command Injection, Server-Side Request Forgery (SSRF), Local File Inclusion (LFI), Directory Traversal, Insecure Direct Object References (IDOR), Open Redirects, CSRF weaknesses, and Security Misconfigurations. It supports authenticated scanning using session cookies, configurable timeout and worker settings, multi-threaded scanning for improved performance, and color-coded severity classification with CVSS scoring to help prioritize security risks. The application generates professional PDF reports for vulnerability assessments, interactive HTML reports for easy analysis, and JSON output for automation and integration with other security workflows. Developed using Python, Requests, urllib3, FPDF2, Matplotlib, JSON, and multi-threading, this project demonstrates practical knowledge of web application security, penetration testing methodologies, vulnerability assessment, and secure software development. The scanner is intended for educational purposes and authorized security testing, providing a practical solution for learning and professional security assessments.
