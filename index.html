Professional OWASP Top 10 Vulnerability Scanner

Python-Based Automated Web Application Security Assessment Tool

Modern web applications are continuously exposed to security threats, making regular vulnerability assessments essential for identifying and mitigating potential risks before they can be exploited. To address this challenge, I developed a professional Python-based vulnerability scanner that automates the detection of common web application security issues aligned with the OWASP Top 10 framework.

The scanner performs comprehensive security testing by identifying vulnerabilities such as SQL Injection, Cross-Site Scripting (XSS), Command Injection, Server-Side Request Forgery (SSRF), Local File Inclusion (LFI), Directory Traversal, Insecure Direct Object References (IDOR), Open Redirects, CSRF weaknesses, and Security Misconfigurations. It supports authenticated scanning using session cookies, configurable timeout and worker settings, multi-threaded scanning for improved performance, and color-coded severity classification with CVSS scoring.

The application generates professional PDF, HTML, and JSON reports for security assessments. Developed using Python, Requests, urllib3, FPDF2, Matplotlib, JSON, and multi-threading, this project demonstrates practical experience in vulnerability assessment, web application security testing, and secure software development. This tool is intended for educational purposes and authorized security testing only.

Deep Learning-Based System for Crop Pest and Weed Identification

Vision Transformer (ViT)-Based Deep Learning for Plant Health Classification

In precision agriculture, timely and accurate identification of pests and weeds is critical for maximizing crop productivity and minimizing economic losses. Traditional methods such as manual inspection or rule-based image analysis are often labor-intensive, slow, and prone to human error.

To address these challenges, I developed a Vision Transformer (ViT)-based deep learning solution for classifying plant images into pest-infected, weed-dominated, and healthy crops. The pretrained ViT-Base-16 model was fine-tuned using data augmentation techniques including color jitter, Gaussian blur, and random rotation. The final model achieved a test accuracy of 91.77%, demonstrating excellent performance for automated crop health monitoring.
